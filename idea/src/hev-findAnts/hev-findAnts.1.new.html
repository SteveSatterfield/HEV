NAME

hev-findAnts - Find "Army of Ants" NIST/HPCVG visualization commands/tools.

 

SYNOPSIS

hev-findAnts [-lmhnvisdogwa] [<string>]

 

DESCRIPTION

hev-findAnts finds commands or tools in directories specified in the PATH environment variable that are part of the "Army of Ants," small reusable applications NIST/HPCVG uses for generating visualizations. There are currently eight catagories of tools:

HEV
(High End Visualization [NIST])
NETX
(NetworkX Utilities [LANL])
VTK
(Visualization Toolkit [Kitware])
ITK
(Insight Toolkit [Kitware])
SAVG
(Scientific Application and Visualization Group [NIST])
DTK
(DIVERSE Toolkit)
OSG
(Open SceneGraph)
DGL
(Distributed Graphics Library [SGI])
IV
(Inventor/VRML 1.0 [SGI])

The optional <string> parameter is used to narrow the search. It is appended to the catagory prefix before searching. Regular expressions can be used.

Command line options are used to select and control the viewing order of each of these catagories. Paging can be controlled with either the less command (default) or the more command.

 

Options

l
Display pages using less command (default).
m
Display pages using more command.
h
List HEV (High End Visualisation) Tools
n
List NETX (NetworkX Utilities) Tools
v
List VTK (Visulaization Toolkit) Tools
i
List ITK (Insight Tookit) Tools
s
List SAVG (Sci. App. Visualization Group) Tools
d
List DTK (DIVERSE Tookit) Tools
o
List OSG (Open Scene Grpah) Tools
g
List DGL (Distributed Graphics Library) Tools
w
List IV (Inventor/VRML) Tools
a
List all tools (default). If this option is used, the listed is ordered as if -hvisdogw had been specified.

 

Examples

To find commands in all catagories with "savg" IMMEDIATLY after the prefix:
    $ hev-findAnts savg

To find HEV commands with "savg" ANYWHERE after the prefix, one needs to put a wildcard before the string:
    $ hev-findAnts -h *savg

To find SAVG and HEV commands with "savg" anywhere after the prefix with any combination of cases, using the 'more' command instead of 'less' for paging:
    $ hev-findAnts -msh *[Ss][Aa][Vv][Gg]

 

NOTES

The 'g' option is used for DGL Tools and 'w' is used for IV Tools.

The order the options specified will determine the order in which the catagories are listed. Only the first occurrence of each option is significant. Repeat occurrences will be ignored.

The 'a' option overides any other catagory selections and orderings.

The 'l' and 'm' options may occur anywhere without changing the order in which the catagories are displayed.
